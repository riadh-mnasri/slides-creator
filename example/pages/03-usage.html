Comment utiliser

Depuis un dossier (trié alphabétiquement) :

node slides-creator.js --input ./mes-pages --output slides.html

Avec des fichiers explicites (ordre libre) :

node slides-creator.js intro.html methode.html conclusion.html

Toutes les options :

--title "Titre de la présentation"
--transition  fade | slide | zoom | convex | none
--slide-number           affiche le numéro de slide
--no-source-styles       ignore le CSS des pages source
--reveal-theme white     surcharge le thème Reveal.js
