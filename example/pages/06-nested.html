Architecture de l'outil

  • Parsing HTML — extraction du contenu avec Cheerio
    • Corps de la page (<body>)
    • Styles inline et fichiers CSS relatifs
  • Scoping CSS — isolation des styles par slide
    • Les règles body / html sont filtrées
    • Sélecteurs préfixés par [data-slide-id]
  • Rendu Reveal.js — présentation HTML autonome
    • Chargé via CDN, aucune dépendance locale pour visionner
    • Thème élégant injecté en priorité
